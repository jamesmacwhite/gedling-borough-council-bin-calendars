Finding your collection schedule

Gedling Borough Council provides a Refuse Collection Days search tool, allowing you to input a street name within the Gedling district which will provide the collection weekday and specific schedule name.

The calendars provided are hosted within a GitHub repository and published through GitHub pages. The "Download iCal" button provides the .ics file of the iCalendar for local use. The "copy to clipboard" button will provide the public URL to the .ics file for subscribing through calendar software or for using as an endpoint in other software that supports the iCalendar format.
