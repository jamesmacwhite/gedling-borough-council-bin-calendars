Gedling Borough Council has a total of 20 different refuse/recycling collections and 10 garden waste collections. Depending on the area within the Gedling district, a specific weekday and schedule name is assigned. Using this search tool, you can find the specific your specific bin collection dates/schedule.

Address search
Uses the newer Gedling bin collection day search which gets bin collections by address and uses live data.
Street search
Uses the older refuse search to search general street names to map to calendar schedules.

About this data

This search tool uses data from Gedling Borough Council with an open-source API to provide the data in consumable formats for display on this website as well as other applications. You can read more about the API behind this search from the main repository.
