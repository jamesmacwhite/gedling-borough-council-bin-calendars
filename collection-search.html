Gedling Borough Council provides 20 different refuse/waste collections and 10 garden waste collections. Depending on the area within the Gedling district, a specific weekday and schedule name is assigned. Using this search tool, you can find the specific collection information for bin collections.

You must enter a street name only, no property numbers or postcode information.

{% include find-collection-schedule.html %}

Some street names will return multiple collections, due to being on a boundary. When multiple results are returned, the area and in some cases property numbers are shown to help identify the correct collection. This additional data is provided by Gedling Borough Council.

About this data source

This search tool uses data from Gedling Borough Council and the official refuse collection days search. The search result format provided by apps.gedling.gov.uk does not provide the collection day and name directly, so this search tool takes all the available data, while linking directly to collection calendars from search results.
