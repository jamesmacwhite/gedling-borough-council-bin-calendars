Finding your collection schedule

Gedling Borough Council provides a Refuse Collection Days search tool, allowing you to input your street name within the Gedling district area which will provide your collection weekday and schedule.

The calendar are hosted on GitHub, the "Download iCal" button provides the raw iCalendar data which you can save locally, the copy to clipboard button provides the path to .ics file for subscribing or using in other software.
